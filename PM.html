화이트 테마 통합 레이아웃 / 설비별 KPI는 YTD와 선택월 기준으로 표시됩니다.

파일을 선택하면 설비별 카드가 자동으로 갱신됩니다.
엑셀 파일을 업로드하면 각 설비가 하나의 화이트 테마 레이아웃에 표시됩니다.
엑셀 구조 기준
- 시트명: 월간실적
- 연도: D5:L5 / 월: D6:L6 (L6 = YTD)
- Conveyor: D7:L9 / ASRS: D10:L12 / Electrical: D13:L15 / Utility: D16:L18 가정
- 상단 전체 요약(누적 계획, 누적 실적, 평균 달성률, Best Equipment)은 제거했습니다.
- 각 설비 상단 KPI는 YTD + 선택한 종료월 데이터를 함께 표시합니다.
